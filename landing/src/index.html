How It Works

SentinelStacks makes it easy to create, run, and manage AI agents

1

Create an Agent

Define your agent using YAML or natural language. Specify capabilities, tools, and memory requirements.

sentinel agent create --name terraform-agent \
--description "Manages cloud infrastructure" \
--model llama3 --memory vector
2

Run Your Agent

Execute your agent in interactive mode or as part of automated workflows.

sentinel agent run --name terraform-agent --interactive
3

Share and Collaborate

Push your agents to the registry to share with your team or the community.

sentinel registry push --name terraform-agent
