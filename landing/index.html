Installation

Install SentinelStacks using your preferred method:

                                
# Using Docker
docker run -v ~/.sentinel:/root/.sentinel sentinelstacks:latest agent run terraform-agent

# Using Binary
curl -sSL https://get.sentinelstacks.io | sh

# Using Package Manager
brew install sentinelstacks  # macOS
apt install sentinelstacks   # Ubuntu/Debian
